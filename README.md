## üì¶ Installation

### Prerequisites
- Python 3.10+
- Node.js 16+

### Quick Start

1. **Clone and Setup**
```bash
git clone <repository-url>
cd wms-mvp
```

2. **Backend Setup**
```bash
cd backend
pip install -r requirements.txt
python app.py
```

3. **Frontend Setup**
```bash
cd frontend
npm install
npm start
```

4. **Environment Variables**
```bash
cp docs/.env.example .env
# Edit .env with your API keys
```
